Guide to using ANAL_qPCR_template_20191212:

- two identical tables side by side

- Fill in Ct values to the pink cells
- Each gene is in triplicates (for biological triplicates)
- 1st gene is the housekeeping gene
- Rename the gene names

- Coulmns represent different treatment conditions
- First one is always control

- use '2^-()' row values for populating ~/Data/Templates/log2_ddCt_data.csv value column
